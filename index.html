Begleitapp

Marktoptimierer als eigenstaendige Webapp

Diese Version arbeitet nur mit einer exportierten marktoptimierer-shared.json aus der Haupt-App. Keine API, keine Datenbank, keine Plenty- oder eBay-Anbindung.

Bitte eine exportierte JSON-Datei laden.

Optional kann eine gebuehrenrechner-daten.json fuer lokale Gewinnberechnungen geladen werden.

Im Tabellenbereich kann horizontal und vertikal gescrollt werden.
